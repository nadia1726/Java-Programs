Java Assignment: Installment Calculator Program

Objective
To create a monthly installment calculator in Java using the `Scanner` class for user input and basic arithmetic operators for calculations.

Problem Statement
Write a Java program that takes the price of a product (e.g., a phone) and the total number of installments from the user as input. The program should then calculate and display the exact amount the user needs to pay per month.

 Requirements
1. Use the `Scanner` class to handle user input.
2. Prompt the user to enter the `phone price` (Integer type).
3. Prompt the user to enter the `Number Of Installments` (Integer type).
4. Apply the correct mathematical logic/operator to calculate the monthly installment.
5. Display the final output clearly on the console, appended with `Euros/Taka`.
6. Properly close the `Scanner` object to prevent memory leaks.

Sample Input/Output

Sample 1:
Enter your phone price: 1200
Enter Your Number Of Installment: 6
Installment PerMonth = 200 Euros/Taka
